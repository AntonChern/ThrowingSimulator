# 3D Top-Down Sandbox

# Description
The player controls a capsule-shaped character and interacts with the environment. The main task is to manipulate objects on the level.

# Game Modes
* Single-player mode: the player completes the level alone, without the need for an internet connection.
* Multiplayer mode: the opportunity to play with other players online. Framework Colyseus was used to implement multiplayer.

# Game Mechanics
* Character control: the character moves around the level using the WASD keys.
* Interaction with the environment: there are boxes of various sizes (from 5 to 8) on the level. The player can pick up boxes using the E key and throw them.

# Getting Started
To start the game, follow these steps:

* Run the game executable file (or open the project in Unity).
* In the main menu, select the desired game mode: single-player or multiplayer.
